# QuickPay

### Prerequirements

 - Install  [nodejs](https://nodejs.org/en/Install)
 - Install [git](https://git-scm.com/download)
 
### How to run ?

* **Clone the repo** 
    ```bash
    git clone https://github.com/VladCherniavsky/QuickPayTool.git
    ```
* **Enter the folder**
     ```bash 
     cd QuickPayTool
     ```
     
* **Install dependencies**
```bash
  npm install
     ```
     
* **Run project**
    bash  
    npm start
    ```
